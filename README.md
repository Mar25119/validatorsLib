# Проект: Классификация строк с помощью validators

Учебный проект, демонстрирующий использование библиотеки validators для определения типа данных: email, URL или IP-адрес.

## Задача
Автоматически определить, представляет ли строка корректный email, URL, IP-адрес или неизвестный формат.

##  Требования
- Python 3.7+
- Библиотека validators:
 
Bash
  pip install validators
  
##  Запуск
Bash
python validator_tool.py
Пример вывода:
' user@example.com' ? email
' https://example.com' ? url
'192.168.1.1' ? ip
'2001:db8::1' ? ip
'not-an-email' ? unknown

##  Почему это полезно?
- Избегаем ошибок при работе с пользовательским вводом
- Упрощаем логику валидации без регулярных выражений
- Повышаем надёжность и читаемость кода

>  Подходит для использования в ботах, веб-формах, CLI-инструментах и учебных проектах.
