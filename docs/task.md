# Техническое задание

## 1. Описание задачи
Разработка утилиты, которая принимает список строк и классифицирует каждую как: email, URL, IP-адрес или "неизвестный формат". Для определения типа используется библиотека `validators`.

## 2. Цель работы
Освоить базовое применение библиотеки `validators` для классификации и валидации пользовательских данных.
## 3. Функциональные требования
- Реализовать функцию classify_input(text: str) -> str, возвращающую одну из меток: "email", "url", "ip", "unknown".
- Поддерживать IPv4 и IPv6.
- Программа должна выводить результат для каждого элемента тестового списка.

## 4. Нефункциональные требования
- Код должен быть читаемым и легко расширяемым.
- Использовать только библиотеку validators (без регулярных выражений).
- Обработка должна быть безопасной: не должно возникать исключений при некорректном вводе.

## 5. Ограничения и допущения
- Не требуется проверять доступность URL или существование email.
- Все входные данные — строки.
- Библиотека validators установлена (pip install validators).

## 6. Источник ТЗ
Самостоятельная постановка.
